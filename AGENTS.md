# Git Guidelines
- Вы можете изменять, создавать и тестировать код в рабочей директории.
- IMPORTANT: NEVER commit or push to the repository without explicit instruction from the human. (Никогда не коммитьте и не пушьте без прямого указания человека).
- Вы должны только вносить изменения в файлы и предоставлять отчет (diff) для ревью.
